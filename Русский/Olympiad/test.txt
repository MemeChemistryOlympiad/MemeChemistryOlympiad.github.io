Для участия в лиге Нубов необходимо соотвествовать следующим критериям:

*Никто из Участников Команды не может быть участником предыдущих Международной или Менделеевской Олимпиад
*Никто из Участников Команды не может обладать дипломом по химии
*Никто из Участников Команды не может быть студентом ВУЗа химического направления
*Никто из Участников Команды не может быть лауреатом Национальной Олимпиады их родной страны или их аналогов в странах вне ЕС: Для РФ-Региональный этап ВсОШ
* Учителям предлагаем участвовать в Лиге Любителей или Профессионалов

Для участия в лиге Любителей необходимо соответствовать следующим критериям:

*Никто из Участников Команды не может быть участником предыдущих Международной или Менделеевской Олимпиад
*Никто из Участников Команды не может обладать дипломом по химии
* Учителям предлагаем участвовать в этой Лиге или Лиге Профессионалов
